# Czech Terraria Translation
This translation will be resourcepack replacing English language to Czech.

## How to

We will translate only the right half.
From this:
```  
"Key": {
	"UP": "UP",
	"DOWN": "DOWN"
```
To this:
```  
"Key": {
	"UP": "NAHORU",
	"DOWN": "DOLÅ®"
```
